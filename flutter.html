Nasıl Flutter Mac bilgisayara kurulur?

• Git bağlantısı Flutter için önemli Flutter’ın kendi sayfasına giriyoruz.
Önce Git ayarlarını yapıyoruz.
•Terminali açalım (Command + Space)
• Önce git version kontrolü yapalım.($ git --version)
• Dosya yolunu öğrenelim.($ which git)
• Cihazınızda git kurulu olmadığını düşünelim ve indirmek için aşağıdaki komutu çalıştıralım.
($ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh))

• Flutter’ı güncel web sitesinden indirelim.
• İndirilen dosya zip tipinde değil. Bu sebeple Belgeler kısmına gelip yeni klasör oluşturalım.
• Adına flutter diyelim ve indirilen flutter klasörünü bu dosyanın içine ekleyelim.
• Yeni bir terminal açalım.(Command + Space)
• ($ echo $SHELL) komutunu çalıştıralım.
• Bendeki cihazda /bin/bash geri dönüşü aldım fakat kurulum işlemlerine bash üzerinden devam ettiğimde istediğim sonucu alamadım.
• Bu sebeple komut alanına ($ vim .zhrc) diyerek devam edelim.

• Açılan ekranda export PATH=“$PATH: yazalım
• PATH alanına flutter > flutter > bin klasörünün path ini verip çift tırnak koyalım çünkü indirdiğimiz flutter klasörünü belgeler içinde açtığımız fluter klasörü içine koyduk.
-Belgeler
--flutter
---flutter
şeklinde bir dizinimiz var.
• export PATH="$PATH:/Users/merve/Documents/flutter/flutter/bin" şeklinde dizini tanımlıyoruz.
• Açılan ekrandan esc tuşuna basıp insert alanına :wq! yazalım ve bu ekrandan çıkalım.
• Yeni bir terminal açalım.(Command + Space)
• (Flutter —Version) komutunu çalıştıralım.

Nasıl Flutter Mac bilgisayarda Android Studio ile entegre edilir?

• Flutter Mac cihazda kurulu olduğunu varsaydığımız Android Cihaz ile birlikte çalışır.
Entegrasyon işlemleri için öncelikle nje durumda olduğumuzu görelim.
•Terminali açalım (Command + Space)
• ($ Flutter doctor ) komutunu çalıştırıp eksiklerimizi görelim.
• Önerilen Android Lisans onaylama alanında verilen komutu çalıştıralım ve hatamızı alalım.

• ($ vim .zhrc) komutunu çalıştıralım.
• Bu ekrana gelmemizin sebebi Android SDK Manager dosya yolunu kaydetmemiz gerektiğidir.
• Android Studio açılış ekranında more actions butonuna tıklayıp, SDK Manager seçelim.
• zshrc dosyası içine export PATH=“$PATH:/Users/merve/Library/Android/sdk/platform-tools" dosya yolunu ekleyelim.
• Yeni bir terminal açalım.(Command + Space)
• (flutter doctor ) komutunu çalıştıralım.

• Android Studio açılış ekranında more actions butonuna tıklayıp, SDK Manager seçelim.Android SDK Command Line tools install diyelim.
• Yeni bir terminal açalım.(Command + Space)
• (flutter doctor ) komutunu çalıştıralım.
• Önerilen komutlar arasındaki Android Licences komutunu çalıştıralım.

How to run Flutter project in XCode?

I will write very soon about this topic.

Oluşturulmuş bir Flutter projesini XCode'da nasıl çalıştırırım?

• Flutter projesini cihaza koplayalım.
• Projenin içinde varsa ios dosyasını başka yere taşıyoruz.
• Bu işlem için XCode'da Signing alanında Team ve Bundle identifier alanı girilmiş olmalıdır.
• 'Key Chain Access' erişimi gerekmektedir, bunun için cihaz paraloları talep edilir.
• Yeni bir terminal açalım.(Command + Space)
• 'flutter create --platform ios .' komutunu çalıştıralım.
• Gerekli pod dosyaları ve diğer dosyaları bu komut ile kendi oluşturuyor.
• flutter build ios komutu ile proje yapılandırılıyor.
• Proje dosyasını çalıştıralım.
